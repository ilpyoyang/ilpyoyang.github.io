우리의 피땀눈물을 줄여줄 수 있는 윈도우 OS 기준으로 코드를 작성할 때 주로 사용하는 유용한 단축키를 정리했습니다.
+ ```Ctrl```+```Shift```+```F``` ------------------------ 전체 찾기
+ ```Alt```+```Enter``` ------------------------------- Quick Fix
+ ```Ctrl```+```Space``` ------------------------------ 자동완성
+ ```Ctrl```+```/``` ------------------------------------ 주석처리
+ ```Ctrl```+```Alt```+```O``` --------------------------- import문 정리
+ ```Ctrl```+```Alt```+```M``` --------------------------- 메서드 추출
+ ```Alt```/```Ctrl```+```Shift```+```↑```/```↓``` ----------- 코드 위치 변경
+ ```Alt```+```↑```/```↓``` -------------------------------- 세로줄 다중커서
